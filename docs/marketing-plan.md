# 📢 마케팅 기획서

## 싸인해주세요 (SignPlease)

> **버전**: 2.0  
> **작성일**: 2026년 2월 3일  
> **브랜드 철학**: 사장님의 하루에서 하나의 걱정을 지우는 브랜드

---

## 1. 브랜드 정체성

### 한 문장

> **"싸인해주세요는 사장님의 하루에서 하나의 걱정을 지우는 브랜드다."**

### 브랜드가 하는 일

| 하지 않는 것           | 하는 것                  |
| ---------------------- | ------------------------ |
| 설명하지 않는다        | 그냥 없앤다              |
| 가르치지 않는다        | 걱정을 지운다            |
| 복잡하게 만들지 않는다 | 생각 안 해도 되게 만든다 |

### 핵심 가치

| 가치       | 의미                        | 언어화             |
| ---------- | --------------------------- | ------------------ |
| **빠름**   | 생각이 들면 바로 끝난다     | "1분이면 됩니다"   |
| **간결함** | 불필요한 선택지와 설명 제거 | "이것만 누르세요"  |
| **안심**   | AI 검토, 기록 보관          | "다 알아서 됩니다" |

### 톤앤매너

```
❌ 하지 않는 말
─────────────────────────
"표준근로계약서를 작성하실 수 있습니다"
"AI 노무사가 법적 검토를 진행합니다"
"과태료 500만원의 위험이 있습니다"

✅ 하는 말
─────────────────────────
"1분이면 끝나요"
"그냥 이렇게 하면 됩니다"
"걱정 마세요, 다 알아서 돼요"
```

### 슬로건

> **"사장님 걱정 하나를 지웁니다"**

대안:

- 계약, 생각하지 마세요
- 1분이면 충분합니다

---

## 2. 마케팅 원칙

### 하지 않는 것

| 원칙                  | 이유                                            |
| --------------------- | ----------------------------------------------- |
| **공포 마케팅 안 함** | "과태료 500만원" 같은 겁주기 → 걱정을 늘림      |
| **기능 나열 안 함**   | "AI 노무사 + 전자서명 + 클라우드 보관" → 복잡함 |
| **전문가 행세 안 함** | 법률 용어, 조항 인용 → 사장님과 거리감          |
| **FOMO 안 씀**        | "지금 아니면", "마감 임박" → 불안 조장          |

### 하는 것

| 원칙       | 방법                  |
| ---------- | --------------------- |
| **공감**   | "계약서, 미루게 되죠" |
| **안심**   | "다 알아서 됩니다"    |
| **단순함** | 한 문장으로 끝냄      |
| **담백함** | 과장 없이 사실만      |

## 3. 타겟 & 인사이트

### Primary: 소규모 요식업 사장님

```
┌─────────────────────────────────────────────────────────────┐
│  사장님의 하루                                               │
│  ─────────────────────────────────────────────────────────  │
│                                                             │
│  07:00  가게 오픈 준비                                       │
│  08:00  재료 체크, 청소                                      │
│  11:00  점심 영업 시작                                       │
│  15:00  잠깐 쉬는 시간 ← 📱 SNS 볼 때                        │
│  17:00  저녁 영업 준비                                       │
│  22:00  마감, 정산                                          │
│  23:00  퇴근 후 ← 📱 SNS 볼 때                               │
│                                                             │
│  "계약서? 중요한 거 알아. 근데 바빠서 자꾸 미루게 돼."       │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 핵심 인사이트

> **계약서는 '해야 하는데 미루는 일'이다.**  
> 어렵고, 귀찮고, 나중에 문제 될까 불안하다.

이 불안을 없애는 게 우리 역할이다.

## 4. 플랫폼별 전략

### 4.1 쓰레드 (Threads)

**역할**: 공감과 안심을 주는 짧은 이야기

#### 콘텐츠 원칙

- 한 포스트에 하나의 메시지만
- 3줄 이하로 끝냄
- 설명하지 않음

#### 예시

**공감형**

```
계약서, 미루고 있죠.
저도 그랬어요.
근데 1분이면 끝나더라고요.
```

```
"나중에 써도 되지 않아요?"
라고 물으면
사실 지금 쓰는 게 나아요.
그게 끝이에요.
```

```
계약서 쓴다고
하루가 복잡해지진 않아요.
1분이면 끝나니까.
```

**안심형**

```
계약서 양식 검색하다
30분 날린 적 있죠.

이제 안 해도 돼요.
```

```
이게 맞나 싶은 불안,
이제 없어요.
AI가 알아서 체크해요.
```

**담백형 (브랜드)**

```
싸인해주세요.

사장님 걱정 하나를 지웁니다.
```

#### 운영

| 항목     | 내용                           |
| -------- | ------------------------------ |
| 빈도     | 주 3-4회                       |
| 시간     | 오후 3시, 밤 11시              |
| 해시태그 | 최소화. #사장님 #계약서 정도만 |

## 5. 배너 광고

### 원칙

- 한 문장만
- 기능 설명 안 함
- 불안 조장 안 함
- 여백 많이

### 카피

**메인 카피**

| 버전 | 카피                        |
| ---- | --------------------------- |
| A    | 사장님 걱정 하나를 지웁니다 |
| B    | 계약, 1분이면 끝나요        |
| C    | 계약서, 생각하지 마세요     |
| D    | 계약 걱정 없는 하루         |

**서브 카피** (필요시)

| 메인                        | 서브           |
| --------------------------- | -------------- |
| 사장님 걱정 하나를 지웁니다 | 싸인해주세요   |
| 계약, 1분이면 끝나요        | 지금 시작하기  |
| 계약서, 생각하지 마세요     | 다 알아서 돼요 |

### CTA 버튼

- "시작하기" (간결)
- "1분 만에 시작" (시간 강조)
- "걱정 지우기" (감정 연결)

### 하지 않는 배너

```
❌ 이런 거 안 함
─────────────────────────
"AI 노무사가 검토하는 표준근로계약서"
→ 너무 길고, 기능 나열

"과태료 500만원, 피하세요!"
→ 공포 마케팅, 걱정을 늘림

"12,847명이 선택!"
→ 우리 브랜드 톤과 안 맞음

"지금 가입하면 5건 무료!"
→ FOMO, 급하게 느껴짐
```

## 7. 성과 측정
